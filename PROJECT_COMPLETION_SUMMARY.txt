================================================================================
                            CONFIGURATION NEEDED
================================================================================

Before building, configure:

1. Bundle Identifiers (3 places)
   - Main App: com.yourcompany.pocketfence
   - Network Extension: com.yourcompany.pocketfence.NetworkExtension
   - App Group: group.com.yourcompany.pocketfence

2. Development Team
   - Select in both targets (main app + extension)

3. Capabilities
   - Network Extensions (packet-tunnel-provider)
   - App Groups (matching identifier)
   - Local Network (device detection)

4. Optional: AdMob Configuration
   - Replace test ad unit IDs in Constants.swift
   - Add GADApplicationIdentifier to Info.plist

5. Optional: In-App Purchase
   - Create product in App Store Connect
   - Update product ID in Constants.swift
